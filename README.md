**💬 Real-Time Chat App | Java Swing ⚙️ + Socket.IO 🚀**
A real-time cross-platform chatting application built using Java Swing for the GUI and Socket.IO for fast, bi-directional communication. This project demonstrates how Java-based desktop applications can interact seamlessly with modern web socket servers — perfect for learning real-time systems, UI design, and socket-based networking.

🔥 Key Highlights
✨ Java Swing GUI – Clean, responsive, and beginner-friendly UI
⚡ Socket.IO Integration – Real-time messaging over WebSockets
🌐 Backend – Lightweight Socket.IO server
👥 Multi-Client Support – Connect multiple Java clients to one server
📦 Modular Structure – Easy to read, modify, and expand

🧠 Tech Stack
Layer	Technology
Frontend	Java Swing (GUI)
Networking	Socket.IO (WebSockets)
Backend	Node.js + Express
Client Comm.	Java Socket.IO Client
